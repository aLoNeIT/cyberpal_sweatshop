# 安全规范

本文档用于统一接口参数处理和响应数据脱敏要求。

详细登录态、权限、白名单、输入输出安全、文件安全和敏感信息保护要求见 `14-permission-and-security-standards.md`。

## 1. 参数安全

- 接收前端参数时必须先过滤。
- 接收前端参数后必须进行类型、长度、枚举、必填和数据范围校验。
- 禁止将未经白名单过滤的前端字段直接用于查询、排序、更新或删除条件。

## 2. 响应安全

- 响应数据必须脱敏。
- 脱敏逻辑必须使用统一脱敏函数处理。
- 响应数据必须按接口场景最小化返回，禁止直接返回未筛选的模型原始数据。
- 日志中的敏感信息过滤要求见 `15-exception-logging-standards.md`。
